Terminal 美化

如果你是 bash 的使用者，可以考慮使用 zsh 這個 shell 來提昇你的 terminal 體驗，以下都是使用 zsh 來操作的。

install zsh

假如你還沒有安裝 zsh ，那你可以使用 sudo apt install zsh 指令來安裝（假如你是 Ubuntu 或 Debian使用者）。

接著，你可以使用 zsh 指令來啟動你的 zsh 。

第一次啟動 zsh 他會幫助你完成一些 zsh 的設定，只要照做就好了。
